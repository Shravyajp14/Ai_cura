AICura – AI-Powered Healthcare Engine  

AICura is an AI-powered healthcare system designed to assist patients and doctors with **disease prediction, medication tracking, diet planning, and doctor recommendations**.  
It uses **Machine Learning (SVM)** models for predicting diseases and integrates user-friendly features for better healthcare management.  

---
Features  

- Disease Prediction
  - Heart Disease  
  - Liver Disease  
  - Diabetes  

- Medication Tracking – Patients can log and manage their prescribed medicines.  

- Diet Recommendations – AI-driven suggestions for healthy diet plans.  

- Doctor Recommendations – Suggests certified doctors based on condition.  

- Doctor Certification – Ensures authenticity of healthcare professionals.  

- Future Scope  
  - OCR integration: Upload PDF reports and automatically extract patient data  
  - Expanded prediction models (cancer, kidney disease, etc.)  
  - Mobile-friendly application  

---
  Tech Stack Used

- **Frontend:** React.js, TailwindCSS  
- **Backend:** Flask / Django (Python)  
- **Machine Learning:** Python (Scikit-learn, Pandas, NumPy)  
- **Database:** MySQL / MongoDB  
- **Tools:** Figma (UI/UX), Git, Jira
